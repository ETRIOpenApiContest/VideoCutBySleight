장면 분할 API는 동영상의 장면 분할 요청 결과를 JSON 형태의 Text 데이터로 반환


#정상 응답
[HTTP Respone Header]
Access-Control-Allow-Origin:*
Connection:close
Content-Length:50783
Content-Type:application/json; charset=UTF-8

[HTTP Respone Body]
{
	"request_id": "reserved field",
	"result": 0,
	"return_type": "com.google.gson.internal.LinkedTreeMap",
	"return_object": {
		"file_id": {비디오 파일 파일 ID}
	}
}

#오류발생
[HTTP Respone Header]
Access-Control-Allow-Origin:*
Connection:close
Content-Length:0
Content-Type:application/json; charset=UTF-8

[HTTP Respone Body]
{
	"request_id": "reserved field",
	"result": -1,
	"reason": {오류 메시지}
}

#정보
장면 분할 기본정보 / file_id (JSON Key 이름) / 장면분할 처리를 위한 file_id
